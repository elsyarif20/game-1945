Cara Menggunakan:
Buat repositori baru di GitHub Anda untuk game ini (misalnya dengan nama dirgantara-1945).
Masukkan file index.html dari kode sebelumnya.
Buat file baru bernama README.md dan masukkan seluruh teks dari blok kode di atas ke dalamnya.
Ganti teks username-anda pada bagian clone URL dengan username GitHub Anda yang sebenarnya.
Repositori Anda sekarang siap dan terlihat sangat profesional!
